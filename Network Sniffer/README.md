# About the script

A Python based Network Sniffer Tool using Scapy <br>
Syntax: 

           sudo python3 Sniff33r.py [-d/-s/-p] PORT | tee OUTPUT.TXT
Usage:    
           
           sudo python3 Sniff33r.py -p 80 | tee log.txt
