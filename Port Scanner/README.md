# About Port Scanner
A simple Port scanner using socket lib
<br>Syntax:

    sudo python3 Port_scanner.py TARGET_IP START_PORT END_PORT THREAD

Usage:
    
    sudo python3 Port_scanner.py 10.10.10.10 0 65535 100
